# Online Betting Dashboard - Backend

---

## Installation

1. Install dependencies:
   ```sh
   yarn install
   ```

2. Set up environment variables:
    - Copy `.env.example` to `.env` and configure database connection.

## Running the Backend

### Development Mode
```sh
yarn dev
```

### Production Mode
```sh
yarn build && yarn start
```

## Database Seeding
```sh
yarn seed
```
